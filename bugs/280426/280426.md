# Mobak Login Automation Tests

## Objetivo
Automatizar o fluxo de login da plataforma Mobak utilizando Robot Framework.

## Pré-requisitos
- Python 3.8+
- Robot Framework
- SeleniumLibrary

## Instalação
pip install robotframework robotframework-seleniumlibrary

## Execução
robot login_tests.robot

## Resultados
Os relatórios são gerados em:
- report.html (visão executiva)
- log.html (detalhes passo a passo)

## Estrutura BDD
Os testes seguem o padrão Gherkin (Given/When/Then)
